## üé∂ Deskripsi Proyek

Aplikasi web pencarian musik, album, dan artis yang dibangun menggunakan **ReactJS** dan **iTunes Search API**. Proyek ini bertujuan untuk memenuhi semua kriteria UTS mata kuliah [Pengembangan Aplikasi Web RB].

Fitur utama yang diimplementasikan untuk memenuhi kriteria wajib meliputi:

| Fitur | 
| :--- | 
| Form Pencarian dengan 5 Input|
| Tabel Dinamis dengan Artwork|
| CSS Styling Responsif (Grid/Flexbox, Media Queries) |
| Functional Components (Minimal 4), useState, useEffect, Props |
| Modern JavaScript (Arrow Functions, async/await, Destructuring)| 
| API Integration (Fetch Data, Loading State, Error Handling) |
| Audio Preview Player dan Playlist Builder dengan localStorage |

## üöÄ Instalasi dan Menjalankan Proyek

Pastikan Anda sudah menginstal **Node.js (LTS)** di komputer Anda.

1.  **Clone Repository:**
    ```bash
    git clone [Masukkan Link GitHub Repository Anda]
    cd music-explorer
    ```
2.  **Instal Dependensi:**
    ```bash
    npm install
    ```
3.  **Jalankan di Lingkungan Lokal:**
    ```bash
    npm run dev
    ```
    Aplikasi akan berjalan di `http://localhost:5174/` (atau *port* lain yang tersedia).
